# ADR Scaffolding Command (`awf new adr`)

Implements [ADR-0042](../decisions/0042-adr-scaffolding-command.md). Design rationale lives there;
this plan is the execution record only.

## Goal

Give `awf-proposing-adr` a deterministic `awf new adr "<title>"` command that computes the next ADR
number, copies the rendered `docs/decisions/template.md` with its marker comments stripped and its
date/title heading filled in, and refuses to overwrite an existing file — replacing the free-hand
number-pick/copy/strip-markers steps an agent currently does by hand.

## Architecture summary

- `internal/adr` (currently parse/render-only) gains `NextNumber` and `NewFile`, plus a `slugify`
  helper and a `now` time seam.
- `internal/project.Project` gains an exported `NewADR` method wrapping those two, mirroring the
  existing `CheckInvariants`/`Audit` facade pattern — `cmd/awf` never reaches `Layout` fields
  directly.
- `cmd/awf` gains a `new` top-level command (`awf new <kind> <title...>`, `kind == "adr"` only
  today), gated by the existing ADR-0039 `gate(root)` check like `sync`/`check`/`invariants`/
  `audit`/`list`.
- The shipped `awf-proposing-adr` skill template is edited to invoke the new command instead of
  describing the manual steps.
- Doc-currency updates (`internal/adr` package doc, `docs/architecture.md`'s `internal/adr/` bullet,
  AGENTS.md's gated-command invariant text, the `tooling` and `adr-system` domain narratives) land
  alongside, and ADR-0042 flips to `Implemented` in the final commit (design and implementation land
  together, per the ADR).

  ```go

  // now returns the current time; overridden in tests for deterministic
  // date-stamping (mirrors cmd/awf's getwd/isInteractive seam pattern).
  var now = time.Now

  // markerLineRe matches an awf-injected marker comment line, so NewFile can
  // strip every one — the GENERATED banner and each per-section awf:edit
  // provenance pointer — from a copied template.
  var markerLineRe = regexp.MustCompile(`(?m)^<!-- (GENERATED by awf|awf:edit).*-->\n`)

  // slugNonAlnumRe collapses everything but lowercase letters and digits into a
  // single hyphen for slugify.
  var slugNonAlnumRe = regexp.MustCompile(`[^a-z0-9]+`)

  // NextNumber returns the next available 4-digit ADR number for dir: one more
  // than the highest number ParseDir finds, or "0001" for an ADR-less dir.
  // invariant: adr-new-sequential-numbering
  func NextNumber(dir string) (string, error) {
  	adrs, err := ParseDir(dir)
  	if err != nil {
  		return "", err
  	}
  	max := 0
  	for _, a := range adrs {
  		n, err := strconv.Atoi(a.Number)
  		if err != nil { // coverage-ignore: a.Number is always a 4-digit numeral matched by fileRe
  			return "", err
  		}
  		if n > max {
  			max = n
  		}
  	}
  	return fmt.Sprintf("%04d", max+1), nil
  }

  // slugify lowercases title and collapses every run of non-alphanumeric
  // characters into a single hyphen, trimming leading/trailing hyphens. A title
  // with no alphanumeric characters is a hard error — there is no reasonable
  // filename to derive.
  func slugify(title string) (string, error) {
  	s := slugNonAlnumRe.ReplaceAllString(strings.ToLower(title), "-")
  	s = strings.Trim(s, "-")
  	if s == "" {
  		return "", fmt.Errorf("adr: title %q has no usable characters for a filename", title)
  	}
  	return s, nil
  }

  // replaceOnce replaces the first occurrence of old in s, erroring if old is
  // absent so a drifted template (a renamed or missing placeholder) fails loudly
  // instead of shipping an unresolved placeholder into a committed ADR.
  func replaceOnce(s, old, replacement string) (string, error) {
  	if !strings.Contains(s, old) {
  		return "", fmt.Errorf("adr: template missing expected %q", old)
  	}
  	return strings.Replace(s, old, replacement, 1), nil
  }

  // NewFile scaffolds a new ADR under dir: the next sequential number, the
  // rendered template.md with every marker comment stripped and its date and
  // title heading filled in, named NNNN-slug.md. Refuses to overwrite an
  // existing file at that path.
  // invariant: adr-new-strips-markers
  // invariant: adr-new-heading-matches-file
  // invariant: adr-new-no-overwrite
  func NewFile(dir, title string) (string, error) {
  	title = strings.TrimSpace(title)
  	number, err := NextNumber(dir)
  	if err != nil {
  		return "", err
  	}
  	slug, err := slugify(title)
  	if err != nil {
  		return "", err
  	}
  	path := filepath.Join(dir, number+"-"+slug+".md")
  	if _, err := os.Stat(path); err == nil { // coverage-ignore: NextNumber always returns one more than every existing NNNN-*.md file, so this path can only pre-exist via a concurrent-process race a single-threaded test cannot construct
  		return "", fmt.Errorf("adr: %s already exists", path)
  	} else if !os.IsNotExist(err) { // coverage-ignore: Stat fails here only on a permission fault a test cannot trigger
  		return "", err
  	}
  	raw, err := os.ReadFile(filepath.Join(dir, "template.md"))
  	if err != nil {
  		return "", fmt.Errorf("adr: read template: %w", err)
  	}
  	content := markerLineRe.ReplaceAllString(string(raw), "")
  	content, err = replaceOnce(content, "date: YYYY-MM-DD", "date: "+now().Format("2006-01-02"))
  	if err != nil {
  		return "", err
  	}
  	content, err = replaceOnce(content, "# ADR-NNNN: Title", fmt.Sprintf("# ADR-%s: %s", number, title))
  	if err != nil {
  		return "", err
  	}
  	if err := os.WriteFile(path, []byte(content), 0o644); err != nil { // coverage-ignore: post-Stat write into a dir just read by NextNumber/ParseDir; fails only on a permission fault a test cannot trigger
  		return "", err
  	}
  	return path, nil
  }
  ```
