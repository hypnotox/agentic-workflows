## Current state

Decisions live as append-only ADRs under `docs/decisions/`, parsed by `internal/adr` (a shared `internal/frontmatter` parser) into a generated `ACTIVE.md` status index — regenerated by `awf sync` and drift-checked by a regenerate-and-compare path. Each ADR declares `domains:`, which drives this doc's generated `## Decisions` index (same regenerate-and-compare mechanism). The workflow chain (brainstorm → plan/ADR → review → impl → review) is enforced by the `awf-*` skills and reviewer agents. The ADR guide (`README.md`) and skeleton (`template.md`) are awf-managed singletons (ADR-0021), so the `.layout.adrReadme` / `.layout.adrTemplate` references resolve in any adopter project.
